# 🎵 音軌編輯應用專案

## 📝 專案簡介
基於 [[DDD]] 分散式架構的音軌編輯應用，目標是為音樂初學者與創作者提供簡單易用的合作編輯平台。

## 🎯 目標
- MVP 階段使用簡單方式實現核心功能，未來升級至 Rust + WebGPU。
- 提供 MIDI 編輯與音效處理，並支援 WebGL 高效渲染。
- 支援多用戶協作與即時音軌編輯。

## 📅 時程規劃
- [ ] 2025 Q1 完成 MVP 核心功能。
- [ ] 2025 Q2 測試與迭代，用戶回饋優化。

## ✅ 進度追蹤
- 目前使用 Blazor + Web Workers 進行多執行緒實作。
- 已設計 MIDI Piano Roll 編輯界面與音頻軌道拖動功能。
